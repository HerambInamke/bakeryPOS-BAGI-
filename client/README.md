This is a Vite + React + TailwindCSS frontend scaffold for the SaleSync landing page and app UI.

PowerShell quickstart:

```powershell
cd client; npm install; npm run dev
```

- Dev server: `npm run dev` (Vite)
- Build: `npm run build`
- Preview build: `npm run preview`

The landing page is implemented in `src/App.jsx`.

Notes:
- Tailwind is already configured. After `npm install`, `npm run dev` should start the Vite dev server on a local port (e.g. http://localhost:5173).
- The page title is set in `index.html`.
